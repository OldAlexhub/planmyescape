# PlanMyEscape - React Travel Planning App

PlanMyEscape is a React-based travel planning app that helps users plan their trips, budget expenses, manage itineraries, and keep track of their travel plans. The app integrates with a Node.js and Python backend to fetch and store data, and it also connects to MongoDB for persistent data storage.

## Features

- **User Authentication:** Signup and login functionality.
- **Vacation Planning:** Plan trips, including budget estimation and accommodation costs.
- **Full Itinerary Management:** View and customize trip details, such as flight and transportation costs.
- **Bookkeeping:** Manage travel expenses by adding, viewing, and deleting entries.
- **Secure Routes:** Restrict access to certain routes based on user authentication.

## Technologies

- **Frontend:** React, Axios, Bootstrap
- **Backend:** Node.js, Express, Python (Flask), MongoDB
- **Hosting:** Render

## Installation

To set up this project locally, follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) and npm (or [Yarn](https://yarnpkg.com/))
- [React.js](https://reactjs.org/)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) or a local MongoDB instance
- A Python backend setup (Flask)

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/OldAlexhub/planmyescape.git
   cd planmyescape
   ```
